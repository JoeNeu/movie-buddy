# Movie-Buddy

## Spring Boot Backend

### Installation

Install Maven or use IntelliJ
https://maven.apache.org/download.cgi

Add Maven Project and run Step mvn:install
IntelliJ: right click on pom -> +Add as Maven Project

    mvn install
    
    mvn spring-boot:run
    
    running at http://localhost:5000
